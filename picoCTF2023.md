#### UnforgottenBits

Searched through files. Found browsing-history in home folder and some links in it. Found some strange looking pictures (prob AI). Then found IRC logs and found something was hidden with steghide and I got a password for it. Used it on pictures I found in gallery dir. The logs also said about openssl, AES and cbc encryption and there were salt, key and iv for it. Nothing interesting found. Tried it with other files and for one file given password was wrong. There was no password from stegseek and rockyou.txt. Decided to find deleted emails, we have learnt from emails. Found them with autopsy and we see that someone recommends out user's files to follow password philosophy from https://xkcd.com/936/. Password we found for bmp files were 4 league of legends characters and by logs we can see our user was playing league of legends so password for 7.bmp should also be 4 league of legends champions. In one file 3.txt we found "I keep forgetting this, but it starts like: yasuoaatrox...", so we need to guess 2 other characters from league of legends. Made a python script and bruteforce password. Password was: yasuoaatroxashecassiopeia. We got encrypted file and tried to decode it with old info. It didn't work. Found some strange text and decoded it with golden ratio decode, because we found it in old files. It was right. Found the flag in it.
